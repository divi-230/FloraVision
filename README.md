## Setup & Run

### Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/divi-230/FloraVision.git
   ```

2. Navigate to the project folder:
   ```bash
   cd FloraVision
   ```

3. Open `index.html` in your browser or use the **Live Server** extension in Visual Studio Code.
